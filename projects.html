Huffman Encryptor

This project uses Python and Flask to allow users to upload a text file which the application then puts through a Huffman Encryption algorith which returns an encrypted file as well as an encrypted key. When both the key and encrypted file are put back into the app the original text is returned. Read More about it Here

Social Stocks

Social Stocks is a web app which allows users to search for stock prices and compare current stock prices to what they paid. This web app was made using Django, sqlite, and the alphavantage stock API. You can visit the github page for the website Here.
